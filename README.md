## Stack:
1. React
2. Redux Toolkit
3. Redux Persist
4. React Tostify
5. Json Server
6. Axios
7. SCSS

## Installation
```
git clone https://github.com/Reyrin/contacts.git
```
## Installing dependencies
```
npm install
```
## Starting a project
```
npm run server
npm run start
```
## Building a project
```
npm run build
```
